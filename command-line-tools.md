
# sydCreateDatabase

Usage: `sydCreateClinicDatabase <schema> <filename.db> <folder>`. It create a database (sqlite file) name `filename.db` associated with a folder where the images will be stored. The filename and the folder must be in the same folder. 

The `schema`is `StandardDatabase`. Other types of schema will be propose in the future. Schemas are plugin libraries that can be dynamically found in the `SYD_PLUGIN` folder.
